# OmarVellouis.com - Astro Website

Modern, fast, and SEO-optimized multipage website built with Astro for digital marketing consultant business.

## 🚀 Project Structure

```
omarvellouis-site/
├── public/              # Static assets (images, favicon, etc)
├── src/
│   ├── components/      # Reusable components
│   │   ├── Header.astro
│   │   └── Footer.astro
│   ├── layouts/         # Page layouts
│   │   └── BaseLayout.astro
│   ├── pages/           # File-based routing
│   │   ├── index.astro        # Homepage
│   │   ├── konsultasi.astro   # Consultation landing page
│   │   ├── tools/             # Tools pages
│   │   ├── case-studies/      # Case studies
│   │   ├── frameworks/        # Framework pages
│   │   └── blog/              # Blog posts
│   └── styles/          # Global styles
├── astro.config.mjs     # Astro configuration
└── package.json
```

## 🎯 Key Features

- ✅ **Lightning Fast**: Static site generation for maximum performance
- ✅ **SEO-First**: Built-in meta tags, Open Graph, Twitter Cards
- ✅ **Mobile Optimized**: Responsive design, optimized for Samsung A55 5G
- ✅ **Brand Consistent**: Plus Jakarta Sans typography, green (#0F9D58) primary color
- ✅ **Analytics Ready**: Facebook Pixel integration built-in
- ✅ **Multipage Ready**: File-based routing for easy page creation

## 📦 Setup Instructions

### 1. Install Dependencies

```bash
cd omarvellouis-site
npm install
```

### 2. Development Server

```bash
npm run dev
```

Visit `http://localhost:4321`

### 3. Build for Production

```bash
npm run build
```

Output will be in `dist/` folder

### 4. Preview Production Build

```bash
npm run preview
```

### Update Facebook Pixel ID

Edit `src/layouts/BaseLayout.astro`:

```javascript
fbq('init', 'YOUR_PIXEL_ID'); // Replace with actual pixel ID
```

### Update WhatsApp Number

Edit `src/pages/konsultasi.astro`:

```html
href="https://wa.me/6281234567890?text=..." 
<!-- Replace with your WhatsApp number -->
```

### Update Social Links

Edit `src/components/Footer.astro` to update Instagram, Threads, LinkedIn URLs

## 📄 Adding New Pages

Astro uses file-based routing. To add a new page:

1. Create a new `.astro` file in `src/pages/`
2. The filename becomes the URL path

Examples:
- `src/pages/about.astro` → `omarvellouis.com/about`
- `src/pages/blog/post-1.astro` → `omarvellouis.com/blog/post-1`

## 🚀 Deployment

### Option 1: Vercel (Recommended)

1. Push code to GitHub
2. Connect repository to Vercel
3. Vercel auto-deploys on every push
4. Free SSL, CDN, automatic optimization

### Option 2: Netlify

1. Push code to GitHub
2. Connect repository to Netlify
3. Build command: `npm run build`
4. Publish directory: `dist`

### Option 3: Cloudflare Pages

1. Push code to GitHub
2. Connect to Cloudflare Pages
3. Build command: `npm run build`
4. Output directory: `dist`

## 📝 DNS Configuration

Point your domain to deployment platform:

**For Vercel:**
- A record: `76.76.21.21`
- CNAME: `cname.vercel-dns.com`

**For Netlify:**
- Check Netlify dashboard for specific DNS records

**For Cloudflare Pages:**
- Cloudflare automatically handles DNS if domain is on Cloudflare

## 🔍 SEO Checklist

- ✅ Unique title & description per page
- ✅ Open Graph tags for social sharing
- ✅ Canonical URLs
- ✅ Alt text for images (add when uploading images)
- ✅ Structured data (consider adding JSON-LD)

Astro automatically:
- ✅ Generates static HTML (no JavaScript by default)
- ✅ Optimizes images (when using `<Image>` component)
- ✅ Bundles and minifies CSS
- ✅ Tree-shakes unused code
